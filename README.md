# Bank Customer Database Project (SQL)

# Project Overview
This project demonstrates a **Bank Customer Management System** using **SQL**.  
It includes database design, table creation, data insertion, and complex SQL queries
to analyze customer accounts, transactions, and relationships.
This project is useful for SQL practice, interviews, and academic submissions.

# Database Schema
The database contains the following tables:
- BANK_CUSTOMER`
- BANK_CUSTOMER_EXPORT`
- BANK_ACCOUNT`
- BANK_ACCOUNT_DETAILS`
- BANK_ACCOUNT_RELATIONSHIP_DETAILS`
- BANK_ACCOUNT_TRANSACTION`
- BANK_CUSTOMER_MESSAGES`
- BANK_INTEREST_RATE`
